# tetris-cpp-sdl2

My Tetris clone in C++ with SDL2.

# Library Using
  * SDL2
  * SDL2_image
  * SDL2_ttf
 
# Installation (on Linux)
  * Install library
  * Download the source code
  * Compile:
  - cd src
  - make
  - ./main
  
# Controls
  * [Left-Arrow]/[Right-Arrow] - Move piece left/right
  * [Down-Arrow] - Move piece one row down
  * [A]/[D] - Rotate piece counter-clockwise/clockwise
  * [Space] - Drop piece
